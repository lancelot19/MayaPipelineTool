"# MayaPipelineTool" 

### INSTALL ###

- Placer le dossier où vous le souhaitez.

- Ouvrir Maya.env.

- créer la variable PYTHONPATH et la parametrer comme suit :

	PYTHONPATH = ;P:\td\pipelineTool_v0.1.0;P:\td\pipelineTool_v0.1.0\venv27\Lib\site-packages

- Enregistrer Maya.env puis ouvrir Maya.

- Ouvrir le script editor, créer une nouvelle fenêtre 'Python' grâce au '+'.

- Drag/Drop le fichier 'code_maya_appel_interface_Qt.py' dans la nouvelle fenêtre Python du script editor.

- Selectionner le code puis Drag/Drop dans un shelf maya pour créer un bouton.
